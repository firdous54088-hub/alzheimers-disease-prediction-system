# Alzheimer's Disease Prediction System

An AI-powered web application that predicts Alzheimer's disease stages from MRI brain scans using Deep Learning.

## Features

- Upload MRI Brain Scan
- AI-based Alzheimer's Prediction
- Confidence Score
- Risk Assessment
- Probability Distribution
- Patient Information
- PDF Report Generation
- Modern Bootstrap UI

## Technologies Used

- Python
- Flask
- TensorFlow / Keras
- NumPy
- Bootstrap 5
- HTML
- CSS
- JavaScript
- ReportLab

## Installation

Clone the repository.

```bash
git clone https://github.com/YOUR_USERNAME/alzheimers-disease-prediction.git
```

Move into the project.

```bash
cd alzh-main
```

Create virtual environment.

```bash
python -m venv env
```

Activate it.

Windows

```bash
env\Scripts\activate
```

Install dependencies.

```bash
pip install -r req.txt
```

Run the project.

```bash
python app.py
```

Open browser

```
http://127.0.0.1:5000
```
